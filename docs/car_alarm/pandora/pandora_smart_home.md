    - PanDECT X-1700 BT
    - PanDECT X-1900 BT
    - Pandora DX-4G
    - Pandora DX-4G S
    - Pandora DX-4GL
    - Pandora DX-90BT + NAV-10
    - Pandora DX-9X LoRa + NAV-10
    - Pandora DXL-4970
    - Pandora DXL-5570
    - Pandora NAV-08 (маячок)
    
Для развертывания требуется Home Assistant Community Store (HACS)
