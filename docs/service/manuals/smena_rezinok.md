# Смена резинок у родных дворников

Есть несколько вариантов:

1. Снять наконечники и заменить резинки
2. Модифицироавть дворник, чтобы менять резинки без снятия наконечников

??? info "Подробное описание варианта 2 (без снятия наконечников)"

    [Оригинальная инструкция на 2dvornikov](https://2dvornika.ru/install-dntl22-rezinka.html)

    Производитель данной модели стеклоочистителя - щетки - дворника в основе использовал ("позаимствовал") схему и конструктив от щетки пр-ва DENSO (впервые использованной на LEXUS NX в 2014 году), но в отличии от DENSO, к сожалению не предусмотрел отверстий для замены резинки. Для начала надо вырезать (**будьте ОЧЕНЬ осторожны с канцелярским ножом**) аналогичные отверстия как показано на фотографии ниже (пластик достаточно "мягкий-податливый") и уже затем вставить новую резинку!
